Strings as Immutable Objects

 

Immutable Objects

Strings are immutable objects. That means once a string has been created with the new operator or by assignment of a string literal, the object CAN NOT change. That is because the String class has no mutator methods. The String reference variable can be changed to reference a different string object, but original object can not be changed.

Given this String definition:

String city= "Cupertino";

the String reference variable city points to a String containing the characters "Cupertino".

 

After this statement:

city = "Saratoga";


city references a new String containing the characters "Saratoga". The old String still exists. It was not changed. But there is no longer a reference to it.
