Using an Array

 

Now let's look again at the problem of reading 12 monthly high temperatures from the user and finding the average high temperature for the year. This time we will use an array, and for extra practice, we will also find how many times the average temperature occurs in the array.

TempsArray

import java.util.Scanner;
/**
   Shows the multiple values each as a separate variable
   @author Kathleen O'Brien
   @version 1.1
*/
public class TempsArray
{
   public static void main(String[] arg)
   {
      //declare variables and constants
      final int NUMBER_OF_TEMPS = 12;
      int total = 0;
      int average = 0;

      //make a Scanner
      Scanner console = new Scanner(System.in);

      //define an array to hold temperatures
      int[] temperatures = new int[NUMBER_OF_TEMPS];

      //fill array (good place to use a for loop)
      for (int i = 0; i < temperatures.length; i++)
      {
         //we add 1 so that the first number requested is for month 1 not 0
         System.out.println("Enter the temperature "
            + "for month " + (i + 1));
         //we save the input at the current index in the array
         temperatures[i] = console.nextInt();
      }

      /*calculate the average
        notice use of the array's length
        notice we start with i = 0
        We could use an enhanced for loop
      */
      for (int i = 0; i < temperatures.length; i++)
      {
         total = total + temperatures[i];
      }

      average = total / NUMBER_OF_TEMPS;


      System.out.println("The average high temperature is "
         + average);

      //find how many times the average temperature actually occurred
      int count = 0;
      for (int temperature : temperatures)
      {
         if (temperature == average)
         {
            count++;
         }
      }

      System.out.println("The number of times " + average
         + " occurred is " + count);

   }//end main method
}//end class

 

The last for loop is an example of a linear search.
