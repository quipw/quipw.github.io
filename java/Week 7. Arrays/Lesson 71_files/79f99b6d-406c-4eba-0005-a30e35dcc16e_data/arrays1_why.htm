Why Arrays?

 

Definition of Array

Arrays are a way of collecting together a set of data of the same type. An array is a data structure. You will learn more about data structures in CIS27C. Like about everything else in Java, arrays are objects.

The formal definition: An array is a sequence of values of the same type.

 

Why Use Arrays?

Suppose we want to ask the user for the high temperature for each month of the year and then find the average high temp for that year. Using what we know so far, we have to use twelve variables called temp1, temp2, ... through temp 12.

 int temp1;
 . . . 
 int temp12;

We will need to have twelve separate prompts and statements to read each values.

System.out.println("Enter the temperature for month 1: ");
temp1 = console.nextInt();
   .
   .
   .
System.out.println("Enter the temperature for month 12: ");
temp12 = console.nextInt();

 

To find the total of the temperatures, calculate and print the average:

final int NUMBER_OF_MONTHS = 12;
total = (temp1 + temp2 + temp3 + temp4
  + temp5 + temp6 + temp7 + temp8 + temp9
  + temp10 + temp11 + temp12);


average = total / NUMBER_OF_MONTHS;
System.out.println("The average high temperature is "
  + average);

TempsNotArray.java

import java.util.Scanner;
/**
   Get the values for high temperature for a year
   Use a separate variable for each value
   @author Kathleen O'Brien
   @version 1.1
*/
public class TempsNotArray
{
   public static void main(String[] arg)
   {
      final int NUMBER_OF_TEMPS = 12;
      int total = 0;
      int average = 0;

      //make a Scanner
      Scanner console = new Scanner(System.in);

      //define 12 variables to hold temperatures
      int temp1;
      int temp2;
      int temp3;
      int temp4;
      int temp5;
      int temp6;
      int temp7;
      int temp8;
      int temp9;
      int temp10;
      int temp11;
      int temp12;

      //Get the values for temps
      System.out.println("Enter the temperature for month 1: ");
      temp1 = console.nextInt();

      System.out.println("Enter the temperature for month 2: ");
      temp2 = console.nextInt();

      System.out.println("Enter the temperature for month 3: ");
      temp3 = console.nextInt();

      System.out.println("Enter the temperature for month 4: ");
      temp4 = console.nextInt();

      System.out.println("Enter the temperature for month 5: ");
      temp5 = console.nextInt();

      System.out.println("Enter the temperature for month 6: ");
      temp6 = console.nextInt();

      System.out.println("Enter the temperature for month 7: ");
      temp7 = console.nextInt();

      System.out.println("Enter the temperature for month 8: ");
      temp8 = console.nextInt();

      System.out.println("Enter the temperature for month 9: ");
      temp9 = console.nextInt();

      System.out.println("Enter the temperature for month 10: ");
      temp10 = console.nextInt();

      System.out.println("Enter the temperature for month 11: ");
      temp11 = console.nextInt();

      System.out.println("Enter the temperature for month 12: ");
      temp12 = console.nextInt();


      total = (temp1 + temp2 + temp3 + temp4
         + temp5 + temp6 + temp7 + temp8 + temp9
         + temp10 + temp11 + temp12);
      average = total / NUMBER_OF_TEMPS;

      System.out.println("The average high temperature is "
         + average);

   }//end main method
}//end class

 

This is doable with 12 values, but what if we wanted to find the average for 10 years? That would require 120 variables.

Arrays are a data structure designed to handle this type of problem.
