Strings

Strings are a very important data type in any programming language. In Java, strings are not primitive data types. They are objects. (We will talk more about objects later in the quarter). There is a class named String and things like "Hello, Java Lover" are objects of that class.

A String is a sequence of characters in double quotes. "Hello, Java Lover" is String literal just like 7 is numeric literal.

We have already seen how to create a String object variable and assign a String object to it.

 

Creating String Variables

This creates a box called message. It can hold a reference to a String. It is a String variable reference.

String message;

 

This creates the String, Hello, and puts its location in the box, message.

message = "Hello "

 

Concatenating Strings

When we concatenate strings with + operator, we are joining the two strings together.

String message = "Hello, ";
String name = "Fred";
String greeting = message + name;

greeting now contains the String "Hello, Fred"

We can also concatenate numbers and other things to a string. Whenever you use + operator and one of the operands is a string, the other is converted to a string. Look at this example

String message = "The answer is ";
int value = 42;
message = message + value;
System.out.println(message);

The output is:

The answer is 42

 

String Methods

length

We can find the number of characters in a String using the length method.

String city = "Saratoga";
int length = city.length();

length now contains the value 8. 

toUpperCase

toUpperCase returns a String that is all upper case.
