The Software
Getting the Software

You can do all of your assignments in the Foothill open lab, where the necessary software is already installed. But since this is an online class, I assume you will want to work from home. To do that, you will need to download and install the following free software:

  • Java Development Kit (JDK)
  • Eclipse (You can use some other development environment if you want but Eclipse is the only supported environment)

Whatever development environment you use, you must first download and install the JDK.

Note for Mac users: You must use OSX. The JDK is already installed as part of that operating system. You just need to download Eclipse.

Downloading the JDK

  • Go to this Oracle Website
  • Select the first option:  Java Platform (JDK) 7
  • Click the license agreement box. Click on your platform.
  • Save to your hard drive.
  • Installation instructions for Windows is at this URL: http://www.oracle.com/technetwork/java/javase/index-137561.html

Downloading Eclipse

Eclipse is a free open source Integrated Development Environment (IDE) available for both the PC and the Mac as well as other operating systems. The nicest thing about it, is that it does syntax checking as you enter your code.

  • Go to this this URL: eclipse.org web site/
  • Click the big, yellow-orange Download Eclipse button at the right
  • Select Eclipse IDE for Java Developers
  • Click your platform under Download Links and follow directions
  • Click Save to Disk in the dialog box
  • Unzip the folder
  • There is no installation required for Eclipse. You may want to put the folder in a convenient location on your hard drive.
  • To run the program you will simply open the folder and double click on eclipse.exe
  • You may want to put a shortcut to eclipse.exe on your desktop or in your dock. You will be using this program a lot.

See the next section for instructions on using Eclipse.
