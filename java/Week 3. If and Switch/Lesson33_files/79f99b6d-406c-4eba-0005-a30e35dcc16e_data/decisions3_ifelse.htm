if/else

 

Sometimes we want to do one thing if an expression is true and something else if it is false. Then we use the if/else construction

if/else Syntax

if (boolean expression)
   statement;
else
   statement;

For example:

if (sunny) 
   go to the beach;
else
   go to a movie;

 

Block Statement

If we wish to execute more than one statement, we can use block statements with curly braces { } for both the if and the else statement. 

if (sunny)
{ 
   go to the beach;
   put on sunscreen;
   wade in the water;
}
else
{
   go to a movie;
}
go to dinner;

 

In either case we will go to dinner.

if/else example 

import java.util.*;

/**
   Shows the use of an if statement
   @author Kathleen O'Brien
   @version 1.1
*/
public class IfDemo
{
   public static void main(String[] arg)
   {
      //make a console reader
      Scanner console = new Scanner(System.in);

      //get input
      System.out.print("Enter an age: ");
      int age = console.nextInt();

      //test the age and print
      if (age > 80)
      {
         System.out.println("That's not so old");
      }
      else
      {
         System.out.println("You are a spring chicken");
      }

      System.out.println("We are done now");
   }//end main method
}//end class

 

Sample output: 

Enter an age: 75
You are a spring chicken
We are done now
Press any key to continue . . .


Enter an age: 90
That's not so old
We are done now
Press any key to continue . . .

Nested if and else

We can have if statements within if statements. Here it becomes even more important to line up the opening and closing braces. If you don't, you are likely to become terminally confused about what is going on. 
if (weekend)
{
   if (sunny)
   {
      go to beach;
   }
   else
   {
      sleep in;
   }
}
else
{
   go to work;
}

 

What will we do if it is sunny on a Monday? It doesn't matter if it is sunny or not. The first thing we check is whether it is a weekend. It is not a weekend, so we look at the else and go to work.
