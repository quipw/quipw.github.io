An Application to Use the Circle Class

 

How to test a class

Now we have a Circle class. What can we do with it? Well, we can compile Circle.java. But we can not execute the resulting class. Go ahead and try it. In Eclipse if you select Run > Run As, you will not see the Java Application option. That's because this is not an application. It does not have a main method. That's okay. Most classes do not have main methods.

To use or test the Circle class, we have to become application programmers and write an application with a main method. Remember that only classes with a main method can be executed. To test our class thoroughly, in the main method of the class, we

  • instantiate a Circle object with the default constructor
  • instantiate a Circle object with the parameterized constructor
  • and call all the methods to see that they work the way we want.

We will look at a partial test here and you will test the class more thoroughly in the homework. We need to document this class, too. We put the obligatory comment at the top of the class. Notice that there are also comments within the code telling you what the code is doing. Put comments inside your code to make it easy for someone else to understand. Use // or /* ...*/ type comments inside your methods. Use /** ...*/ only for the class itself and above methods and constructors.

 

CircleTest 

/**
 * An application to use the Circle class
 * @author Kathleen O'Brien
 *
 */
public class CircleTest 
{
   public static void main(String[] args) 
   {
      //create a default circle
      Circle c1 = new Circle();
      printCircle(c1);
      
      //create a circle with a parameter
      c1 = new Circle(10);
      printCircle(c1);
   }

   /**
    * Prints the information about a Circle
    * @param c the Circle whose information we want to print
    */
   public static void printCircle(Circle c)
   {
      //Tests some methods
      System.out.println("A circle with radius " + c.getRadius());
      System.out.println("   has area "+ c.area());
      
      //use printf to make the output look nice
      System.out.printf("   and circumference %.2f%n",c.circumference() );
   }
}
