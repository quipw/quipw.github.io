Creating Objects

 

How to Make An Object

In Java, we create an object from a class using the new operator. This is the only way an object is created.

Create a Car Object

If we have a class Car, we make a car object like this. (This is called instantiating the object):

new Car();

 

Once we create (instantiate) this car, we can no longer access it because we don't have a "handle" on it. We don't have a reference to it. The new operator creates an object and returns a reference (pointer, address, handle) to it. We can capture the reference in an object reference variable ( sometimes simply called an reference variable or object variable).

 

Create a Car Reference Variable

To create an reference variable, we specify the type (in this case Car) and a name we want to use. For example

Car myHonda;

This creates an object reference variable that can hold a reference to Car. It does not create a Car object. This is important! Car myCar does NOT create an object. An object is only created with the new operator.

 

Create Object and Object Reference Variable in One Statement

We can put these two statements together and create an object and save the reference to it in an object variable

Car myHonda = new Car();

 

Object Reference Variable Declaration and Object Instantiation

 

Car myHonda declares a reference variable that can hold a reference to a Car object. It can not hold a reference to a Scanner object or a Rectangle object. Only a Car object.

new Car() creates an object of type Car. This object has no name. Programmers tend to say "the Car object myHonda." This is not strictly correct. Technically it is "the Car object referenced by the reference variable myHonda"

The = saves the reference to the newly created Car object in the variable myHonda.

 

Creating Other kinds of Objects

The Java libraries are full of classes you can use to create classes. You have used a couple already, Scanner and Random.

Random generator = new Random();

 

Random generator declares a reference variable that can reference an object of type Random. generator can only hold a reference to a Random object. It can not hold a reference to a Car.

new Random() creates a Random object.

The = saves the reference to the newly created Random object in the reference variable generator.

 

You have also created Scanner objects.

Scanner console = new Scanner(System.in);

 

In this case Java needed some extra information (System.in) in order to create the object.

 

Rectangle Class and Objects

Create a Rectangle Object

A Rectangle class exists in the Java library in the package java.awt. A Rectangle object describes a rectangle in 2-D space. We can make a new Rectangle like this:

new Rectangle (100, 150, 50, 60);

 

This creates a rectangle with its upper left hand corner at (100, 150) and with a width of 50 and a height of 60.

 

Create a Rectangle Object Variable

Remember, we can define an object reference variable by giving the type and a name, like this:

Rectangle book;

 

Create Rectangle Object and Object Variable in One Statement

To capture the reference to the object in an object variable

Rectangle book = new Rectangle (100, 150, 50, 60);

 

Create Rectangle Object and Object Variable in Two Statements

This could also be done in multiple steps.

Rectangle book;
book = new Rectangle (100, 150, 50, 60);
