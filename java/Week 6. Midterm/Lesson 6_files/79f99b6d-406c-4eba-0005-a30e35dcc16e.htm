Lesson 6: Midterm and Review
Midterm Review

The midterm will cover chapters 1 - 5. Also Math.pow()

You have 2.5 hours to take the midterm. It can be taken any time during the open window,  but it must be taken in one sitting and submitted by the due date (Check the schedule). No late submissions can be accepted. You have a 4 day window in which to do the exam. If you need it open earlier for some reason, ask me.

Things to know

  1. Declaring variables       
    a. to declare a variable: specify name and data type
    b. name should be in camelCasing
  2. . Declare a constant
    a. use the keyword final
    b. the name should be all uppercase
  3. Assigning values to variables and casting
    a. you can not assign a double to an int unless you cast.
    . int age = (int)4.75
    b. you can not cast to or from boolean
  4. . if, if/else, and if/else if
    a. the else goes with the last unmatched if in the block
    b. look at review question 3.2 and 3.4 at the end of chapter 3
  5. loops: for, while, do/while
    a. do/while loop always executes at least once
    b. sentinel value for while and do/while
    c. the control variable defined in the for loop header is local to the for loop
    it does not exist outside the loop
    d. nested loops
  6. . be able to read a code segment and tell what it prints
  7. . integer division and modulus ( 8/3, 8%3, -8 % 3)
  8. . other operators ++ -- +=
  9. . find errors in code
  10. . Write short programs similar to examples in class or homework
